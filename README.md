# aha
脚手架工具

# 安装
```
npm install aha -g
```
或者
```
git clone https://github.com/zlj002/aha_generator.git

cd aha && npm install

npm link
```

# 使用
在终端中输入 `aha` or `aha -h` , 可以看到帮助信息
```
  Usage: aha <command>


  Commands:

    add|a      新增一个模板
    list|l     现有模板列表
    init|i     生成项目
    delete|d   删除模板

  Options:

    -h, --help     output usage information
    -V, --version  output the version number
```

> 备注如果使用的 `MacOS`, `sudo` 权限是必须的在使用命令 `add` and `delete`的时候.
